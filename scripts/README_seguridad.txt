Medidas básicas de seguridad implementadas:

1. La aplicación utiliza control de acceso mediante usuario y contraseña.
2. Las contraseñas se almacenan cifradas con password_hash().
3. El acceso a determinadas páginas se protege mediante sesiones PHP.
4. La base de datos dispone de copias de seguridad manuales mediante backup.bat.
5. La restauración del sistema se realiza mediante restore.bat.
6. El sistema se ejecuta en entorno local, sin exposición a Internet.
